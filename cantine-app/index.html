Serverless Architectures

Serverless architectures are application designs that incorporate third-party “Backend as a Service” (BaaS) services, and/or that include custom code run in managed, ephemeral containers on a “Functions as a Service” (FaaS) platform.

https://martinfowler.com/articles/serverless.html.

BaaS: Backend as a Service

  • Databases (Firebase, DynamoDB, ...)
  • Auth-Providers (Auth0, ...)
  • ...

FaaS: Function as a Service

  • Code runs in stateless containers
  • event-triggered
  • endless languages
  • endless providers

Example: AWS Lambda

The Serverless Framework is an open-source CLI for building and deploying serverless applications.

https://serverless.com/

serverless framework (core)
